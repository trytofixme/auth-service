### Auth Service (JWT Authorization Service)

**Auth Service** — шаблон микросервиса, отвечающего за аутентификацию и авторизацию пользователей в распределённой системе на основе **JWT (JSON Web Token)**.

#### Основные функции

* Аутентификация пользователей по логину и паролю
* Выпуск **Access Token** и **Refresh Token**
* Обновление access-токенов без повторного ввода учётных данных
* Валидация JWT (подпись, срок действия, claims)
* Управление ролями и правами доступа (RBAC)
* Отзыв токенов (logout, компрометация, смена пароля)
